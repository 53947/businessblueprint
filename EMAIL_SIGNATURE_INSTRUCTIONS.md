## How to Install Your Signature

### Method 1: Use the HTML File (Easiest)

1. **Open `email-signature.html`** in your web browser (Chrome, Safari, Firefox, etc.)
2. **Right-click on the signature image** → Select **"Copy image"**
3. **Paste into your email client** following the instructions below

### Method 2: Direct Installation

#### Gmail:
1. Go to Gmail Settings (gear icon) → **See all settings**
2. Under **General** tab → Scroll to **Signature**
3. Click **Create new** → Give it a name (e.g., "TriadBlue")
4. **Right-click in signature box** → Paste the copied image
5. Click **Save Changes**

#### Outlook (Desktop):
1. File → Options → Mail → **Signatures**
2. Click **New** → Enter a name
3. **Right-click in editor** → Paste
4. Click **OK**

#### Apple Mail:
1. Mail → Preferences → **Signatures**
2. Click **+** to create new
3. Paste the copied image
4. Close preferences (auto-saves)

## Troubleshooting

**Q: Image not showing in my signature?**  
A: Make sure you're copying the IMAGE (not the HTML code). Right-click → "Copy image"

**Q: Can I edit the signature?**  
A: Yes! Edit the original PNG file, save it as `triadblue-signature.png`, and replace the file in `attached_assets/`

**Q: Will this work on mobile?**  
A: Yes! The signature will display correctly on all devices.

**Q: Can multiple people use this?**  
A: Absolutely! Share the `email-signature.html` file or this instructions document.
