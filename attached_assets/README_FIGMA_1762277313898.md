# BusinessBlueprint — Figma Kit (Developer States)
This kit mirrors your Replit HTML and helps you build the **.fig** in minutes with **click-state components**.

## Contents
- `index.dev.html` — click-to-toggle prototype reference
- `index.html` / `styles.css` — static site (already built earlier)
- `tokens.json` — import into Figma (Tokens Studio plugin) to create color/gradient styles
- `assets/` — all icons you provided, embedded-ready

## How to Create the .fig (with developer-oriented component states)
1. Open Figma → **Plugins → Tokens Studio** (free). Import `tokens.json` to create **Color Styles** & **Gradients** under a group named *TriadBlue / BusinessBlueprint*.
2. Create a new **1440 Desktop frame**. Set 12-col grid (80/24 gutter) or your system.
3. Build components:
   - **Pricing Card** component with **variants**: `default` and `selected`. In `selected`, increase brightness by 5% and add a 2px blue outline `#00A3FF`. 
   - **App Tile** component with `default` and `selected` (blue border + glow).
   - **Execution Chip** component with `default` and `selected` (fluorescent edge).
   - **CTA Button** component with `default` and `hover` (optional).
4. Drop icons from `/assets` onto each component.
5. Use **Prototype** tab: set interactions to **On click → Change to → (selected)** with **Smart Animate (200ms, ease out)**.
6. Lay out the page: Header → Flat Icon Grid → Base Plans → Execution → Apps → Coaching → CTA → Dev Schema.
7. Save as **.fig** (File → Save local copy).

## Notes
- ALC and Scale are incompatible (documented in schema).
- Fonts: Archivo Semi Expanded / Archivo.
